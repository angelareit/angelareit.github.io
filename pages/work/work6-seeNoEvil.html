006 : See No Evil

A first person horror adventure game where you have to escape an abandoned basement, while escaping from the unknown. Made using a custom C++ engine, I have worked on the core structure of the game: game logic, controls, rendering, and sound. I have also created the environment assets in Blender.

Tools

  • C++ (OpenGL, Fmod)
  • Photoshop
  • Blender
